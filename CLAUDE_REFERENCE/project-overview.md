- **Project Name:** BORIS Killer (also called "Better BORIS")

- **Brand Positioning:** "Quietly Intimidating Competence"
- **Stage:** First business venture, MVP development phase

---

## What We're Building

A modern construction job management SaaS positioned as a simpler, more user-friendly alternative to BORIS Software. Target market is small to medium construction companies in Australia who are frustrated with BORIS's complexity and steep learning curve.

---

## Core Value Proposition

### Pain → Proof → Payoff Framework:

**Pain (What BORIS Users Hate)**
- Excessive complexity with features most users never need
- Poor user experience and outdated interface
- Integration gaps requiring manual data entry
- Steep learning curve (weeks of training required)
- Expensive pricing jumps ($99/mo → $299/mo with little middle ground)
- Clunky mobile experience

**Our Proof (How We're Better)**
- Clean, intuitive interface requiring minimal training
- Modern UX following current design patterns
- Built-in integrations from day one
- Mobile-first responsive design
- Transparent, flexible pricing
- Faster implementation (days not weeks)

**Payoff (What Users Get)**
- Save time on daily operations
- Reduce errors from manual data entry
- Get teams productive immediately
- Scale pricing with business growth
- Access data anywhere on any device
